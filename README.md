# Dotfiles
---

`hyprland` for tiling window manager<br>
`sddm` for login manager<br>
[lain](https://github.com/Ascaniolamp/Hyprlain) sddm theme<br>
`yay` for aur helper<br>

`waybar` for taskbar<br>
`rofi` for application launcher<br>
`dunst` for notification daemon<br>
`wl-clipboard` for clipboard<br>
`networkmanager` for network daemon<br>
`pipewire` for audio control<br>
`pavucontrol` for gui mixer<br>

`eww` for widgets<br>
`hyprshade`, `swappy`, and `grimblast`/`grim` for screenshoting<br>

`neofetch` for system information<br>
`btop` for system monitoring<br>

`kitty` for terminal<br>
`zsh` for terminal shell<br>
`thunar` for file explorer<br>
`nvim` for text editor<br>

`kvaumtum` and [Wallbash](https://github.com/prasanthrangan/hyprdots/wiki/Wallbash) for qt theme<br>
`bibata-modern-ice` for cursor<br>
`tela-circle-dracula` for icon theme<br>
initial [HyDE](https://github.com/HyDE-Project/HyDE) project install script<br>

```
sudo pacman -Syu && yay -Syu && sudo pacman -S hyprland sddm waybar rofi dunst wl-clipboard kitty thunar kvantum swappy grim pavucontrol pipewire && yay -S neofetch hyprshade grimblast
```
